LockedIn: AI Mobile App for Finding Study Groups

LockedIn mobile app helps college students overcome the challenges of peer collaboration within a course, so they can “lock in” or focus on their studies. The app uses AI to match users with compatible study partners based on survey input of study habits and day/time availability. Students can also create or browse public study sessions to initiate collaboration in their courses. AI also automatically interprets uploaded class materials into actionable tasks, which can be managed through the app.

I designed the structure of our app, including the page layouts and the user experience flow of interaction among screens. We utilized Expo Go for development, and our tech stack included React Native, Convex, and the OpenAI API. We developed the functional fullstack prototype based on our design in about five weeks for an Intro to Human-Computer Interaction course.

Wanderlust: Social Media for Travelers

With the Wanderlust mobile app, travelers can share their experiences by creating personal or public lists of photos categorized by location to document their journeys. Photos marked as public are automatically pinned on an interactive global map. Users can explore a map-based feed, view public posts, and add comments.

Working on a team, I created intuitive user interfaces and implemented key features including Apple Maps integration, location-based pins for user posts, and a modal window for viewing public posts. Using the MERN stack and Apple Maps API, our team developed a fully functional prototype in just one month for the final project of a Social Computing course.

Interactive Data Visualization Webpage: Geographical Movie Trends

This webpage presents movie trends around the world using multiple graphs and an interactive globe interface. Users can explore TMDb movie data through real-time navigation of the 3D Mapbox globe. Dynamic popups and selected filters reveal movie details based on geographic regions.

The webpage was built with JavaScript, HTML, and CSS. Collaborating with a partner for a final project in a Data Visualization course, my largest contribution was the popup feature to display movie information on the globe and developed the filtering functionality.

Ravenswood Reads Website: Custom Logistics Platform for a Literacy Tutoring Program

In this independent project, I took the initiative to propose, design, and develop this internal website to replace disjointed spreadsheets. The platform handles attendance tracking, tutor–tutee pairing, and performance analysis through a scalable data management system. Through iterative feedback, I designed intuitive user flows tailored for the various user roles of student tutors, parents, and program fellows. Given time constraints, I prioritized user experience and core functionality first, with plans to refine the visual design in the near future. Built with React and Firebase, the site is now actively used to improve program efficiency and impact.
